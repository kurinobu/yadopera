# マニュアル 4.8 に「一括登録代行」の記述を追加する 実装計画

**作成日**: 2026年3月14日  
**目的**: ご利用マニュアルの **4.8「CSV一括登録（Standard・Premium）」** に、**有料代行の案内・申し込み導線**を明示的に含め、お仕事につながる部分を利用者が確実に見つけられるようにする。  
**実施結果**: 2026-03-14 に「有料代行をご希望の場合」ブロックおよび料金の目安を追加。続く有料明示・料金提示（`docs/20260314_CSV代行_有料明示と料金提示_調査分析と修正案.md`）で文言を「代行（有料）」に統一し、デプロイ・ステージング確認完了。

---

## 1. 背景と問題

- **現状**: 4.8 節には「対象プラン」「画面の開き方」「使い方（手順）」「CSV形式」「テンプレート」「エラー時の対処」まではあるが、**「代行申し込み」に関する項目が存在しない**。そのため、有料代行を希望する利用者がマニュアル内で申し込み先を見つけられない。
- **経緯**: ロールバック前の Manual.vue には「有料代行をご希望の場合」のブロックが 4.8 に含まれていたが、現在の 4.8 には含まれていない。実装計画で「4.8 に代行申し込みの項目を必須とする」と明文化しておらず、抜け落ちた。
- **方針**: 4.8 の**本文の末尾**に「有料代行をご希望の場合」のブロックを**必ず**追加することを本計画で明示し、再発を防ぐ。

---

## 2. 対象ファイル・場所

| 項目 | 内容 |
|------|------|
| **ファイル** | `frontend/src/views/admin/Manual.vue` |
| **対象** | `sections` 配列のうち、id が `faq` の章（第4章 FAQ管理）に属する、id が `faq-csv-bulk` のサブセクション（4.8 CSV一括登録（Standard・Premium））の **`content`** プロパティ。 |
| **挿入位置** | 既存の 4.8 の `content` の**末尾**。「エラー時の対処（一覧）」の最後の行（「上記以外のメッセージが表示された場合は…」）の**直後**、閉じバッククォート（\`）の**直前に**、新しい見出しと箇条書きを追加する。 |

---

## 3. 追加する文言（案）

以下を 4.8 の `content` の末尾に追加する。**コンテンツ記述ルール**（`docs/マニュアル更新・改訂_必読.md` §4）に従い、箇条書きは行頭に **`- `**（半角ハイフン＋スペース）を付ける。

```
**有料代行をご希望の場合**:
- Standard・Premiumプランの施設では、FAQ管理画面の「CSV一括登録」ボタンの近くに「代行をご希望の方はこちら」リンクが表示されます。左メニューの「CSV代行の申し込み」からも同じ申し込みページへ進めます。
- 申し込みページでは、希望登録件数・希望言語・連絡先・FAQ原稿ファイル（.xlsx / .csv / .txt / .md）などを送信できます。送信内容は運営にメールで届き、料金表に沿った見積もりと決済案内（Stripe連携予定）が行われ、合意後に開発者がCSV一括登録でFAQを登録します。
- 申し込みは [CSV一括登録 代行の申し込み](/admin/csv-bulk-request) から行えます。料金表や詳細は運営までお問い合わせください。
```

- **リンク**: `[表示テキスト](/admin/csv-bulk-request)` の形式とする。`ManualContent.vue` の `formatContent` 内で `[〜](URL)` が `<a href="URL">` に変換されるため、クリックで `/admin/csv-bulk-request` に遷移する。
- **表記の統一**: 「FAQ管理画面の『CSV一括登録』ボタンの近く」「左メニューの『CSV代行の申し込み』」は、現行のフロント実装（FaqManagement.vue の「代行をご希望の方はこちら」、Sidebar.vue の「CSV代行の申し込み」）と一致させる。

## 4. 実装手順（指示があった場合に実施）

| 順 | 作業 | 内容 |
|----|------|------|
| 1 | バックアップ | `frontend/src/views/admin/Manual.vue` を `backups/YYYYMMDD_manual_48_proxy/` などにコピーする。 |
| 2 | 編集 | Manual.vue の 4.8（id: `faq-csv-bulk`）の `content` の、**「上記以外のメッセージが表示された場合は、表示内容に従ってCSVを修正し、再度アップロードしてください」** の直後、閉じバッククォート \` の直前に、上記 §3 のブロック（改行を挟んで）を挿入する。 |
| 3 | 記述ルール確認 | 箇条書きの行がすべて行頭 `- ` で始まっていること、見出しが `**有料代行をご希望の場合**:` の形式であることを確認する。 |
| 4 | 編集後必須手順 | `docs/マニュアル更新・改訂_必読.md` §5 に従い、フロントの再ビルド・再起動、ブラウザのサイトデータ消去・Service Worker 解除・ハードリロードを実施する。 |
| 5 | 確認 | 管理画面で「ご利用マニュアル」を開き、目次から「4.8 CSV一括登録（Standard・Premium）」を選択する。本文の**末尾**に「有料代行をご希望の場合」の見出しと箇条書きが表示され、「CSV一括登録 代行の申し込み」リンクをクリックすると `/admin/csv-bulk-request` に遷移することを確認する。 |

## 5. 完了の定義

- 4.8 の本文に **「有料代行をご希望の場合」** の項目が存在し、**申し込みページへのリンク**が動作すること。
- 利用者がマニュアルのみから、**代行の申し込み先を迷わず見つけられる**状態になること。
