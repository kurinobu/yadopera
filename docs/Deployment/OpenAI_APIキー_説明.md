# OpenAI APIキー 説明

---

## このキーの用途

このOpenAI APIキーは、**やどぺら**のステージング環境で使用されます。

### 具体的な用途

1. **AI対話エンジン**: ゲストからの質問に対してAIが回答を生成
2. **埋め込みベクトル生成**: FAQや質問文をベクトル化して検索
3. **FAQ自動学習機能**: 未解決質問からFAQの回答文を自動生成
4. **信頼度スコア計算**: AI回答の信頼度を評価

### 使用するAPI

- **GPT-4o mini**: 回答生成（コスト効率重視）
- **text-embedding-3-small**: 埋め込みベクトル生成（1536次元）

---

## キー名の推奨

OpenAI Platformでキーを作成する際の名前:

**推奨名**: `yadopera-staging` または `yadopera-staging-2025`

**理由**:
- 用途が明確（やどぺらのステージング環境）
- 環境を識別しやすい
- 後で本番環境用のキーを作成する際に区別しやすい

---

## 注意事項

- このキーは**ステージング環境専用**です
- 本番環境では別のキーを使用することを推奨します
- キーは機密情報なので、他人に共有しないでください
- キーを忘れた場合は、新しいキーを生成できます

---

## 次のステップ

1. OpenAI Platformでキーを作成（名前: `yadopera-staging`）
2. キーをコピー（`sk-`で始まる文字列）
3. Render.comの`OPENAI_API_KEY`環境変数に設定
