# Render.com Pythonバージョン指定方法

**調査結果**: Render.comでPythonバージョンを指定する方法

---

## Pythonバージョン指定方法

### 方法1: 環境変数で指定（推奨）

1. Render.comダッシュボードで、Web Serviceを選択
2. 「Environment」タブを開く
3. 新しい環境変数を追加:
   - **Key**: `PYTHON_VERSION`
   - **Value**: `3.11.8`（または`3.11.9`）

### 方法2: `.python-version`ファイルで指定

1. プロジェクトのルートディレクトリ（`backend/`）に`.python-version`ファイルを作成
2. 内容: `3.11.8`（または`3.11.9`）

### 方法3: `runtime.txt`ファイルで指定

1. `backend/runtime.txt`ファイルを作成
2. 内容: `python-3.11.9`

**注意**: `runtime.txt`の形式は`python-3.11.9`（`python-`プレフィックスが必要）

---

## Python 3.11の利用可能性

Render.comは複数のPythonバージョンをサポートしていますが、利用可能なバージョンは環境によって異なる可能性があります。

**確認方法**:
- Render.comのドキュメントを確認
- または、実際に設定してデプロイを試す

---

## 推奨対応

**方法1（環境変数で指定）**を推奨します。

**理由**:
- 最も確実な方法
- Render.comのドキュメントで推奨されている
- 設定が簡単

---

## 実装手順（方法1を選択した場合）

1. Render.comダッシュボードでWeb Serviceを選択
2. 「Environment」タブを開く
3. 環境変数を追加:
   - **Key**: `PYTHON_VERSION`
   - **Value**: `3.11.8`
4. 保存
5. 再デプロイ（自動的に開始される）

---

## 注意事項

- Python 3.11.8または3.11.9が利用可能かどうかは、実際に設定して確認する必要があります
- 利用できない場合は、Render.comがサポートしている最も近いバージョンを使用してください
