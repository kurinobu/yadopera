# GA4 で A/B の表示数・クリック数・率を確認する手順

最終更新: 2026-04-21 16:20  
対象: `lp_ab_view` / `lp_ab_cta_click` / `lp_ab_assign`

---

## 1. 先に押さえる前提

- LPの計測ID: `G-BE9HZ0XGH4`
- 計測イベント:
  - 表示: `lp_ab_view`（パラメータ `variant=a|b`）
  - クリック: `lp_ab_cta_click`（パラメータ `variant`, `cta_label`）
  - 割当: `lp_ab_assign`
- 最終的な目的:
  - A/B別の表示数
  - A/B別のクリック数
  - A/B別のCTR（クリック数 / 表示数）

### 1.1 現時点の結論（MVP方針）

- 実装は完了しており、イベント送信自体は確認済み。
- 追加実装なし（MVP・大原則準拠）で、既存イベントから A/B 判定を行う。
- まずは標準レポートで全体件数を確定し、`variant` 分解で A/B 件数を取得する。

---

## 2. まず「送信されているか」の確認（ブラウザ）

1. シークレットウィンドウで `https://yadopera.com/?lp=a` を開く
2. DevTools `Network` タブを開く
3. `collect` でフィルタ
4. `collect?v=2` 行を開き、`ペイロード` の `en` を確認
5. `en=lp_ab_view` であることを確認

補足:
- CTAクリック確認は遷移が発生するため、`ログを保持` を ON にして実施する
- `collect` が出ない場合は、拡張機能・広告ブロッカー影響を疑う

---

## 3. GA4 標準レポートで「件数」を確認（推奨）

1. GA4で `レポート` を開く
2. 左メニュー `ユーザー エンゲージメントと…` を開く
3. `イベント` を開く
4. 右上の日付を `今日` に変更して適用
5. イベント一覧テーブル内の検索で `lp_ab_` を入力
6. 以下3行の `イベント数` を確認
   - `lp_ab_view`
   - `lp_ab_cta_click`
   - `lp_ab_assign`

全体CTR:
- `lp_ab_cta_click / lp_ab_view`

---

## 4. A/B別の集計を可能にする設定（初回のみ）

`管理` → `データの表示` → `カスタム定義` で以下を作成:

1. `variant`（イベントスコープ）
   - イベントパラメータ: `variant`
2. `cta_label`（イベントスコープ）
   - イベントパラメータ: `cta_label`

注意:
- 候補に出ない場合でもパラメータ名は手入力で登録する
- 反映まで数時間〜24時間かかることがある

---

## 5. A/B別の表示数・クリック数・率を出す手順

反映後、標準レポートまたは探索で `variant` で分解して確認する。

計算式:
- `CTR_A = lp_ab_cta_click(variant=a) / lp_ab_view(variant=a)`
- `CTR_B = lp_ab_cta_click(variant=b) / lp_ab_view(variant=b)`

### 5.1 標準レポートでの実操作（優先）

1. `レポート` → `ユーザー エンゲージメントと…` → `イベント`
2. 右上日付を `今日` に設定
3. イベント一覧の「表内検索」に `lp_ab_` を入力
4. `lp_ab_view` 行をクリックして詳細へ移動
5. 詳細画面で `variant` 分解を試行
   - 右カード `パラメータ名` で `variant` が選択可能なら利用
   - 選択不可なら、同画面の別導線（表側のディメンション追加）を使う
6. `lp_ab_cta_click` でも同様に `variant` 分解
7. A/B別CTRを計算

---

## 6. つまずきやすい点（今回の実績ベース）

- `DebugView` は出ない場合がある（0件でも送信されているケースあり）
- 送信確認は `collect` の `en` で事実確認する
- 操作案内は画面ラベル差異があるため、**最新スクショを正**として進める
- 複数検索窓があるため、イベント一覧の検索窓とグローバル検索窓を混同しない

2. `lp_ab_cta_click` が見えにくく、CTA遷移時の計測欠落を疑って修正（beacon化・遷移ガード）を実施し再公開。
