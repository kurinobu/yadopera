**不足・不備について**

- 実装計画書に §4.1（マニュアル記載）・§4.2（有料オプション運用開始）・§4.3（フロート申し込み）のタスク・成果物・期待動作が記載されている。

- 申し込みの送信先は **Formspree は使わず、自前 API + Brevo** のみ。ファイル添付は API で受け、Brevo で運営あてに送る。

  **対応**: `backups/20260313_before_rollback/backend/scripts/` および同 `docs/CSV_bulk_registration_function_implementation_plan/` から復元するか、実装計画書 §4.2 の O1・O2・O3 に沿って再作成する。

以上を踏まえ、**計画の記録は揃っており、上記の「Step 2 の復元 or 再作成」を明示すれば、不足している調査・不備はない**と判断できる。
