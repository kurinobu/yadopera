## 2. 週次サマリー統計カード（上から順番）

### 2.1 カード1: 総質問数

**何が表示されているか**:
- ゲストから送信された質問の総数
- 具体的には、過去7日間にゲストがチャットで送信したメッセージ（質問）の件数

**どれくらいの期間**:
- **過去7日間**（今日を含む直近7日間）

**補足説明**:
- この数値は、ゲストが実際に送信した質問の数です
- AIの回答やシステムメッセージは含まれません
- エスカレーションされた質問も含まれます

---

### 2.2 カード2: 自動応答率

**何が表示されているか**:
- AIが自動で回答した質問の割合（パーセンテージ）
- 具体的には、過去7日間の質問のうち、エスカレーションされずにAIが回答した質問の割合

**どれくらいの期間**:
- **過去7日間**（今日を含む直近7日間）

**計算方法**:
- 自動応答率 = (総質問数 - エスカレーション数) / 総質問数 × 100
- 例: 総質問数が100件で、エスカレーションが20件の場合、自動応答率は80%

**補足説明**:
- この数値が高いほど、AIが多くの質問に自動で回答できていることを示します
- 100%に近いほど、スタッフの対応負荷が軽減されていることを意味します

---

### 2.3 カード3: 平均信頼度

**何が表示されているか**:
- AIが回答した際の信頼度の平均値（パーセンテージ）
- 具体的には、過去7日間にAIが生成した回答の信頼度の平均値

**どれくらいの期間**:
- **過去7日間**（今日を含む直近7日間）

**補足説明**:
- 信頼度は、AIが自分の回答にどれだけ自信を持っているかを示す指標です（0%〜100%）
- この数値が高いほど、AIの回答の質が高いことを示します
- 低い場合は、FAQの改善や追加を検討する必要がある可能性があります

---

### 2.4 カード4: 未解決質問

**何が表示されているか**:
- エスカレーションされて、まだ解決されていない質問の件数
- 具体的には、スタッフの対応が必要な状態で、まだ解決されていない質問の数

**どれくらいの期間**:
- **全期間**（期間制限なし）
- 過去のエスカレーションも含む、現在未解決のすべての質問

**補足説明**:
- この数値は、現在対応が必要な質問の数を示します
- 0件に近いほど、すべての質問が適切に解決されていることを意味します
- 数値が高い場合は、スタッフの対応が必要です

## 3. カテゴリ別内訳（円グラフ）

### 3.1 セクション全体

**何が表示されているか**:
- AIが回答する際に使用したFAQのカテゴリ別の使用回数
- 具体的には、過去7日間にAIがFAQを使って回答した回数を、カテゴリ別に集計したもの

**どれくらいの期間**:
- **過去7日間**（今日を含む直近7日間）

**カテゴリの種類**:
1. **Basic（基本情報）**: チェックアウト時間、WiFiパスワードなど、基本的な情報に関するFAQ
2. **Facilities（設備・サービス）**: レンタルバイク、洗濯機など、設備やサービスに関するFAQ
3. **Location（周辺情報）**: 最寄りのコンビニ、観光スポットなど、周辺情報に関するFAQ
4. **Trouble（トラブル対応）**: 鍵をなくした、WiFiが繋がらないなど、トラブル対応に関するFAQ

**補足説明**:
- このグラフは、どのカテゴリのFAQがよく使われているかを視覚的に示します
- 各カテゴリの使用回数が表示され、合計も表示されます
- この情報を参考に、よく使われるカテゴリのFAQを充実させることができます

## 4. まとめ

### 4.1 期間のまとめ

**過去7日間のデータ**:
- 総質問数
- 自動応答率
- 平均信頼度
- カテゴリ別内訳

**全期間のデータ**:
- 未解決質問

### 4.2 ユーザー視点での理解

**宿泊管理者が知りたいこと**:
1. **どれくらい質問が来ているか** → 総質問数（過去7日間）
2. **AIがどれくらい自動で対応できているか** → 自動応答率（過去7日間）
3. **AIの回答の質はどうか** → 平均信頼度（過去7日間）
4. **対応が必要な質問はあるか** → 未解決質問（全期間）
5. **どのカテゴリのFAQがよく使われているか** → カテゴリ別内訳（過去7日間）

## 5. 改善提案

1. **未解決質問の期間が不明確**
   - 現在の表示: 「エスカレーション待ち」
   - 問題: 全期間のデータであることが分からない

2. **カテゴリ別内訳の説明が不正確**

   - 問題: 「メッセージ」という表現が曖昧で、AI回答であることが分からない

3. **自動応答率の説明が不十分**
   - 現在の表示: 「AIが回答した割合」
   - 問題: エスカレーションされなかった割合であることが分からない

1. **未解決質問のサブタイトルを改善**
   - 現在: 「エスカレーション待ち」

   - 現在: 「過去7日間のメッセージで使用されたFAQのカテゴリ集計」
   - 改善案: 「過去7日間のAI回答で使用されたFAQのカテゴリ別使用回数」

3. **自動応答率のサブタイトルを改善**
   - 現在: 「AIが回答した割合」
   - 改善案: 「エスカレーションされずにAIが回答した割合（過去7日間）」
