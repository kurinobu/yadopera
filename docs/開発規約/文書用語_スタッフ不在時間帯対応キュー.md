# 文書・運用・会話での用語 — スタッフ不在時間帯対応キュー

**作成日**: 2026年3月26日  
**目的**: 「夜間キュー」など旧称・略称の再発を防ぎ、**施設管理者向けマニュアル・ヘルプと同じ呼び方**に統一する。

---

## 1. 正しい呼び方（必須）

| 用途 | 用いる語 |
|------|-----------|
| 日本語のドキュメント（`docs/`）、Runbook、引き継ぎ、AI 応答でユーザーに説明するとき | **スタッフ不在時間帯対応キュー** |
| 短くてよいが「夜間」とは言わない表現 | **スタッフ不在時間帯の対応キュー**、**不在時間帯のキュー** |

**禁止（文書・ユーザー向け説明）**: 単独の **「夜間キュー」**、**「夜間対応キュー」**（旧称。新規記述に使わない）。

---

## 2. コード上の名前（事実として残るもの）

実装・DB では歴史的名称が残る。**文書で初出する際は一度だけ対応づけを書く。**

| 種別 | 名称 |
|------|------|
| Python | `OvernightQueueService`、`OvernightQueue` モデル、`overnight_queue` テーブル |
| API パス例 | `/api/v1/admin/overnight-queue/...` |

**書き方の例**: 「スタッフ不在時間帯対応キュー（実装: `OvernightQueueService`）」

---

## 3. AI・執筆者向けチェック

- `docs/` や Runbook を編集したら、**`夜間キュー` を grep しない**（残っていれば **スタッフ不在時間帯対応キュー** に置換するか、§2 のように実装名との対応を添える）。
- ユーザーへの返答でも、**業務用語は「スタッフ不在時間帯対応キュー」に統一**する。
