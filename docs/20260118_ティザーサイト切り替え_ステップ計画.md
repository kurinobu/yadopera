## 6. トラブルシューティング

### 6.1 デプロイが失敗する場合

**原因**: GitHub Actionsの設定エラー

**対処法**:
1. GitHubリポジトリの「Actions」タブでエラーログを確認
2. `.github/workflows/pages.yml` の設定を確認
3. `landing/` フォルダの構造を確認

### 6.2 ティザーサイトが表示されない場合

**原因**: キャッシュの問題

**対処法**:
1. ブラウザのキャッシュをクリア
2. シークレットモードでアクセス
3. GitHub Pagesのデプロイログを確認

### 6.3 フォーム送信が動作しない場合

**原因**: Formspreeの設定エラー

**対処法**:
1. FormspreeのエンドポイントURLを確認
2. フォームの `action` 属性を確認
3. Formspreeのダッシュボードで送信ログを確認
