**チェック項目**:
- [ ] 引き継ぎ書を**最初から最後まで全文精読したか**

## 9. 禁止事項

### 9.1 絶対に禁止される行為

1. **チェックリストを完了する前に作業を開始すること**
2. **引き継ぎ書を全文精読せずに報告すること**
3. **参照文書の内容を引き継ぎ書より優先すること**
4. **解決済みの問題を残存課題として報告すること**
5. **完了済みの作業を再度実施しようとすること**
6. **時刻を確認せずに最新文書を特定すること**
7. **矛盾を確認せずに報告すること**

### 9.2 違反時の対応

**チェックリスト未完了で作業を開始した場合**:
- 即座に作業を停止する
- チェックリストを完了する
- 完了後に作業を再開する

**誤った報告をした場合**:
- 即座に謝罪する
- 原因を調査分析する
- 再発防止策を強化する

## 10. 再発防止策の強化

### 10.1 文書作成時の必須事項

**すべての文書に以下を含める**:
1. **作成日時**: `YYYY年MM月DD日 HH時MM分SS秒`形式
2. **最終更新日時**: `YYYY年MM月DD日 HH時MM分SS秒`形式（更新がある場合）
3. **状態**: 最新の状態を明確に記載

**例**:
```
**作成日時**: 2025年12月17日 21時00分00秒  
**最終更新日時**: 2025年12月17日 21時30分00秒  
**状態**: ✅ **完了**
```

### 10.2 セッション開始時の必須手順

1. **このチェックリストを最初に開く**
2. **セクション1から順番に実行**
3. **すべてのチェック項目にチェックを入れる**
4. **報告形式に従って報告する**
5. **報告完了後に作業を開始する**
