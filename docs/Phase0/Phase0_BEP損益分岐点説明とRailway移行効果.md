## 1. BEP（損益分岐点）とは

### 1.1 定義

**BEP（Break-Even Point）** = **損益分岐点**

**意味**: 営業利益がゼロになる（収益とコストが等しくなる）施設数のこと。

- **BEP未満**: 赤字（コスト > 収益）
- **BEP到達**: 損益分岐（コスト = 収益、営業利益 = ¥0）
- **BEP超過**: 黒字（収益 > コスト）

### 1.2 計算式

```
BEP（施設数） = 月次固定費 / (平均ARPU × 粗利率)
```

**要約定義書v0.2より**:
- 月次固定費: ¥145,000
- 平均ARPU: ¥5,000/月
- 粗利率: 80%
- **BEP = ¥145,000 / (¥5,000 × 0.8) = 36.25施設**

**結論**: **37施設で損益分岐点に到達**（36.25を切り上げ）

### 3.2 BEP計算（Railway使用時）

**前提条件**:
- 月次固定費: ¥121,500（固定費¥1.5k + 運用費¥120k）
  - または ¥120,500（AWS S3無料枠内の場合）
- 平均ARPU: ¥5,000/月
- 粗利率: 80%

**計算**:
```
BEP = ¥121,500 / (¥5,000 × 0.8)
   = ¥121,500 / ¥4,000
   = 30.375施設
```

**結論**: **31施設で損益分岐点に到達**

**AWS S3無料枠内の場合**:
```
BEP = ¥120,500 / (¥5,000 × 0.8)
   = ¥120,500 / ¥4,000
   = 30.125施設
```

**結論**: **31施設で損益分岐点に到達**

### 3.3 BEP減少の意味

**現在の計画（Render.com使用）**:
- BEP: **37施設**
- 意味: 37施設で損益分岐点に到達

**Railway使用時**:
- BEP: **31施設**
- 意味: 31施設で損益分岐点に到達

**減少幅**: 37施設 - 31施設 = **6施設減少**

**意味**:
- **6施設早く黒字化できる**
- より少ない施設数で損益分岐点に到達
- 早期の黒字化が可能

## 6. まとめ

### 6.1 BEP（損益分岐点）とは

- **定義**: 営業利益がゼロになる施設数
- **現在の計画**: 37施設
- **Railway使用時**: 31施設

### 6.2 BEP減少の意味

**「BEPが約4施設減少（36施設 → 32施設）」の意味**:

- **正確には**: 37施設 → 31施設 = **6施設減少**
- **意味**: 
  - より少ない施設数で黒字化できる
  - 6施設早く損益分岐点に到達
  - 早期の黒字化が可能

### 6.3 収益への影響

**Phase 1-3でRailway使用時の効果**:
- **コスト削減**: ¥18,000/月（¥216,000/年）
- **BEP減少**: 37施設 → 31施設（6施設減少）
- **Year 1**: 早期黒字化可能（Year 1 Q4）
- **Year 2**: 営業利益¥216,000/年増加

### 6.4 推奨事項

✅ **Phase 1-3**: Railway Hobby（無料）で開始
- コスト削減: ¥18,000/月
- BEP減少: 6施設
- 早期黒字化可能

✅ **Phase 4**: 利用量増加を確認してからRender.comに移行
- 50施設以上で移行を検討
- 移行コストと収益増加を比較
