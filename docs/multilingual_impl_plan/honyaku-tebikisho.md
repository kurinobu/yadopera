
# 翻訳手引書（宿泊施設向けFAQ）
**ファイル名：honyaku-tebikisho.md**

本手引書は、宿泊施設管理者が **ChatGPTを使ってFAQを正確に翻訳するための実務ガイド** です。  
生成AIを初めて使う方でも **迷わず・安全に・高品質な翻訳** ができるように作成しています。

---

## 1. 目的
- 日本語FAQを **英語・繁体中国語（Traditional Chinese）** に正確に翻訳する
- 誤訳・クレーム・誤解を防ぐ
- インバウンド対応の品質を統一する

---

## 2. 翻訳前の準備（必須）

### 2.1 日本語FAQを整える
翻訳精度は **日本語の書き方で決まります**。

**良い例**
- 1文を短く
- 主語と目的語を明確に
- 曖昧表現を避ける

**悪い例**
- 文が長すぎる
- 「適宜」「なるべく」「状況により」など曖昧語

---

## 3. ChatGPTで翻訳する基本手順（初心者向け）

### Step 1：ChatGPTを開く
- https://chat.openai.com/ にアクセス
- ログインする

### Step 2：翻訳用の指示文（コピペ用）

```
あなたは宿泊業界専門の翻訳者です。
以下の日本語FAQを、自然で正確な英語と繁体中国語に翻訳してください。
宿泊客向けに、丁寧で分かりやすい表現にしてください。
意味は変えず、意訳しすぎないでください。

【日本語FAQ】
（ここにFAQ本文を貼り付け）
```

---

## 4. 翻訳入力例（日本語 → 英語＋繁体中国語）

### 日本語（原文）
```
チェックインは15時から可能です。到着が遅れる場合は事前にご連絡ください。
```

### ChatGPTへの入力例
```
あなたは宿泊業界専門の翻訳者です。
以下の日本語を英語と繁体中国語に翻訳してください。
ホテルの案内として自然な表現にしてください。

チェックインは15時から可能です。到着が遅れる場合は事前にご連絡ください。
```

---

## 5. 期待される翻訳例（正解モデル）

### 英語
```
Check-in is available from 3:00 PM.  
If you expect to arrive late, please inform us in advance.
```

### 繁体中国語（Traditional Chinese）
```
入住時間為下午3點起。  
若您將延遲抵達，請事先通知我們。
```

---

## 6. 翻訳精度を高めるコツ（重要）

### 6.1 1回で完璧を求めない
2回目の指示で改善できます。

**改善指示例**
```
もう少し丁寧な表現にしてください。
より自然なホテル向けの表現に修正してください。
```

---

### 6.2 直訳になりすぎた場合の修正指示
```
直訳ではなく、自然なホテル案内の文章に直してください。
```

---

### 6.3 用語を統一する（例）

| 日本語 | 英語 | 繁体中国語 |
|---|---|---|
| チェックイン | Check-in | 入住 |
| チェックアウト | Check-out | 退房 |
| 宿泊者 | Guest | 住客 |
| フロント | Front Desk | 櫃檯 |

---

## 7. よくある誤訳を防ぐ指示テンプレ

```
意味を変えずに翻訳してください。
宿泊施設の案内として自然な表現にしてください。
法律・規約・注意事項は意訳せず、正確性を優先してください。
```

---

## 8. 確認チェックリスト（必須）

翻訳後、必ず以下を確認してください：

- 日本語の意味と一致しているか
- 日付・時間・料金が間違っていないか
- 丁寧すぎず、失礼でもないか
- 誤解を招く表現がないか

---

## 9. NG例（やってはいけない）

❌ 翻訳を確認せずそのまま公開  
❌ 日本語の意味を変更する  
❌ 文化的に不自然な言い回しを放置  
❌ 数字・料金・時間の誤りを見逃す  

---

## 10. 複数FAQを一括翻訳する方法（CSV想定）

### 推奨フォーマット
| 日本語 | 英語 | 繁体中国語 |
|---|---|---|
| 質問 | （翻訳） | （翻訳） |
| 回答 | （翻訳） | （翻訳） |

### ChatGPTへの指示例
```
以下のFAQ一覧を、英語と繁体中国語に翻訳してください。
表形式で出力してください。
```

---

## 11. トラブル時の対処

| 問題 | 対処 |
|---|---|
| 不自然な翻訳 | 「自然な表現に修正」と指示 |
| 意味が変わった | 原文を短く修正して再翻訳 |
| 専門用語が誤訳 | 用語表を追加して再指示 |

---

## 12. 推奨運用ルール（実務向け）

- **翻訳は必ず人が確認してから公開**
- 規約・免責・料金は **二重チェック**
- 表現の統一ルールを施設内で共有

---

## 13. まとめ

ChatGPT翻訳は非常に便利ですが、  
**「正しい指示」＋「人の確認」＝安全で高品質な翻訳** になります。

本手引書に沿って作業すれば、  
**AI未経験者でも実務レベルの翻訳が可能** です。
