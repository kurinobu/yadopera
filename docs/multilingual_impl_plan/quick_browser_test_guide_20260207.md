## 🐞 よくある問題と対処法

### 問題1: ページが読み込めない
**症状**: `http://localhost:5173` にアクセスできない  
**対処**:
```bash
# フロントエンドサーバーが起動しているか確認
ps aux | grep "vite\|npm run dev"

# ポート 5173 をリッスンしているか確認
netstat -an | grep 5173

# サーバーを再起動
cd /Users/kurinobu/projects/yadopera
npm run dev --host 0.0.0.0
```

### 問題2: セクション 12.2 が見当たらない
**症状**: 翻訳手引書セクションが表示されない  
**対処**:
```bash
# ブラウザキャッシュをクリア
# DevTools > Ctrl+Shift+Delete（またはCmd+Shift+Delete on Mac）

# ページを強制リロード
# Ctrl+F5（または Cmd+Shift+R on Mac）

# ローカルストレージをクリア
# DevTools > Application > Local Storage > Clear All
```

### 問題3: テキストが文字化けしている
**症状**: 日本語が正しく表示されない  
**対処**:
```javascript
// ブラウザコンソールで確認
console.log(document.characterSet); // UTF-8 であることを確認
```

### 問題4: コードブロック内のテンプレートが見えない
**症状**: `` ``` `` 内のコンテンツが表示されない  
**対処**:
