
# 翻訳手引書（宿泊中ゲストFAQ向け）
**ファイル名：honyaku-tebikisho.md**

本書は、**宿泊中のゲストからの質問に回答するFAQを、正確に多言語翻訳するための手引書**です。  
AIや翻訳ツールを使ったことがない方でも、**迷わず運用できる実務マニュアル**として作成しています。

---

# 0. このシステムの目的（最初に理解してください）

**目的：宿泊中のゲストからの質問に、正確かつ誤解なく回答すること**

翻訳対象は、以下のような**「ゲストからの質問」＋「その回答」**です。

例：
- Q. Wi‑Fiは無料ですか？
- Q. タオルはどこで交換できますか？
- Q. 消灯時間は何時ですか？
- Q. 朝食は何時から何時までですか？

---

# 1. 基本構成（必ず「質問 → 回答」の順で作成）

## 正しい形式

### 質問（日本語）
```
朝食は何時から何時まで利用できますか？
```

### 回答（日本語）
```
朝食は午前7時から午前9時まで提供しています。
```

**※ 回答だけを書く運用はNGです。必ず質問とセットで管理してください。**

---

# 2. 翻訳対象言語について（前提）

本システムは**複数言語に対応可能**です。  
本書では**説明しやすさのため、例として「英語」と「繁体中国語」**を使用しますが、  
**実際の運用では他言語（韓国語・フランス語など）にも同じ手順を適用できます。**

---

# 3. 翻訳前の準備（翻訳品質を左右します）

## 3.1 日本語の質問・回答を整える

**良い例**
```
Q. 消灯時間は何時ですか？
A. 消灯時間は午後11時です。
```

**悪い例**
```
夜って何時くらいに電気消す感じですか？
だいたいそのへんです。
```

### 注意点
- 1文を短くする
- 曖昧な言葉を使わない
- 数字・時間・場所を明確にする

# 4. ChatGPTで翻訳する手順（初心者向け）

## Step 1：ChatGPTを開く
https://chat.openai.com/ にアクセスし、ログインします。

---

## Step 2：翻訳用テンプレ（コピペでOK）

```
あなたは宿泊施設向けの翻訳担当者です。
以下の「ゲストからの質問」と「回答」を、
意味を変えずに自然な◯◯語と◯◯語に翻訳してください。
宿泊中のゲストに向けた丁寧で分かりやすい表現にしてください。

【質問】
（ここに日本語の質問を貼る）

【回答】
（ここに日本語の回答を貼る）
```

※ ◯◯語の部分は、必要な言語名（英語・中国語・韓国語など）に書き換えてください。

# 6. 翻訳結果の理想例

## 英語
**Q.**
```

```

```
Breakfast is served from 7:00 AM to 9:00 AM.
```

## 繁体中国語（Traditional Chinese）

早餐供應時間為上午7點至上午9點。

# 7. 翻訳の確認方法（外国語が分からない人向け）

## 7.1 ChatGPTで「逆翻訳」して確認する

### 手順
1. 翻訳された英文・中国語をコピー
2. ChatGPTに以下を入力

```
以下の文章を日本語に戻して意味を確認してください。
（ここに翻訳文を貼る）
```

### チェックポイント
- 日本語に戻した意味が**元の質問・回答と一致しているか**
- 時間・金額・条件が変わっていないか

---

## 7.2 数字・時間・固有名詞を重点確認

| 項目 | 確認内容 |
|---|---|
| 時間 | 7:00 → 9:00 になっていないか |
| 料金 | 無料 → 有料 に変わっていないか |
| 禁止事項 | OK → NG に変わっていないか |

---

## 7.3 不安な場合の再指示テンプレ

```
元の日本語の意味と完全に一致しているか確認し、
ズレがあれば修正してください。
```

---

# 8. よくある失敗と防止策

| 失敗 | 防止策 |
|---|---|
| 意味が変わる | 逆翻訳で確認 |
| 丁寧さ不足 | 「丁寧に修正」と再指示 |
| 宿泊業に不自然 | 「ホテル案内向けに修正」と指示 |

---

# 9. 人による確認とは「外注」ではありません

本システムの想定：
- **翻訳会社に外注する前提ではありません**
- **施設スタッフが意味確認する運用**です

### 確認方法（現実的）
- ChatGPTで逆翻訳
- 数字・条件のチェック
- 社内の日本語理解者が確認

**Premiumで言語数が増えても、外注は不要です。**

---

# 10. 最初に読む簡単クイック手順（初心者向け）

## 3ステップだけ

1. **日本語で「質問」と「回答」を正しく書く**
2. **テンプレをコピペしてChatGPTに翻訳依頼**
3. **逆翻訳で意味がズレていないか確認**

# 11. まとめ

この手引書の目的は  
**「宿泊中のゲストの質問に、正しく答える翻訳を誰でも作れるようにすること」**です。

質問と回答の順序を守り、  
逆翻訳で確認すれば、  
**AI初心者でも安全に多言語FAQを作成できます。**
