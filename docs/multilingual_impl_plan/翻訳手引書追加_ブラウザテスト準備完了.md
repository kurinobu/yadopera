**対処法**:
1. ブラウザのキャッシュをクリア
2. フロントエンドコンテナを再起動
   ```bash
   docker-compose restart frontend
   ```

### 8.2 目次が表示されない場合

**対処法**:
1. ブラウザをリフレッシュ（F5）
2. 開発者ツールのConsoleタブでエラーを確認

### 8.3 セクション番号が正しくない場合

**対処法**:
