## 3. 再発防止策

### 3.1 質問と指示の明確な区別

**ルール**:
1. **質問・意見を聞かれた場合**:
   - まず、質問・意見に対する回答・意見を述べる
   - **「修正しますか？」「実装しますか？」と明示的に確認する**
   - ユーザーの同意を得てから実施する

2. **指示が明確な場合のみ実施**:
   - 「修正してください」「実装してください」などの明確な指示がある場合のみ実施
   - 質問や意見だけの場合は、確認してから実施

### 3.2 確認プロセスの確立

**必須確認項目**:
1. **質問・意見の場合**:
   - [ ] 質問・意見に対する回答・意見を述べた
   - [ ] 「修正しますか？」「実装しますか？」と明示的に確認した
   - [ ] ユーザーの同意を得た

2. **指示の場合**:
   - [ ] 指示内容を理解した
   - [ ] 実施内容を明確にした
   - [ ] バックアップを作成した（必要に応じて）

### 3.3 行動前の確認チェックリスト

**修正・実装を実施する前の必須確認**:
- [ ] ユーザーから明確な指示があったか？
- [ ] 質問・意見の場合は、同意を確認したか？
- [ ] 実施内容を明確にしたか？
- [ ] バックアップを作成したか？（必要に応じて）

**確認が取れていない場合**:
- **修正・実装を実施しない**
- 質問・意見に対する回答・意見を述べる
- 「修正しますか？」「実装しますか？」と明示的に確認する
