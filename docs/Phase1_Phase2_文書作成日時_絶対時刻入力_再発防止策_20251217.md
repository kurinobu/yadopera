## 2. 再発防止策

### 2.1 必須ルール

**ルール**: **全ての文書の作成日時・更新日時には絶対時刻（時分秒）を入力する**

**形式**:
- **日本語形式**: `2025年12月17日 14時24分06秒`
- **英語形式**: `2025-12-17 14:24:06`
- **ISO形式**: `2025-12-17T14:24:06+09:00`

**推奨形式**: **日本語形式**（`YYYY年MM月DD日 HH時MM分SS秒`）

### 2.2 実装方法

#### 2.2.1 時刻取得コマンド

**コマンド**:
```bash
date '+%Y年%m月%d日 %H時%M分%S秒'
```

**出力例**:
```
2025年12月17日 14時24分06秒
```

#### 2.2.2 文書作成時の必須チェックリスト

**文書作成前の必須チェックリスト**:
- [ ] 作成日時に絶対時刻（時分秒）を入力したか

- [ ] 時刻取得コマンドを実行して正確な時刻を取得したか

**文書作成後の必須チェックリスト**:
- [ ] 作成日時に絶対時刻（時分秒）が入力されているか確認
