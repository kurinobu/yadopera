# 施設設定：電話番号のゲスト表示但し書き — 修正案

**作成日**: 2026-03-03  
**目的**: 電話番号を入力・保存するとゲスト画面に表示されることを、施設管理ページで明示する但し書きを追加する。  
**参照**: 大原則（Phase3 現況と残存課題・実装計画）、メールアドレス同様の「ゲスト画面に表示される」旨の案内

---

## 1. 背景

- 施設設定の「電話番号」は任意項目だが、入力して保存すると**ゲストのウェルカム画面（FacilityHeader）にそのまま表示**される。
- メールアドレスには「ゲスト画面にメールアドレスを表示する」のチェックと、「このメールアドレスはゲスト画面に表示されます。……」の但し書きがある。
- 電話番号には同様の説明がなく、ゲストに表示されることが管理者に伝わりにくい。**統一・同一化**（大原則）に従い、電話番号にも「入力して保存するとゲスト画面に表示される」旨の但し書きを追加する。

---

## 2. 大原則との対応

| 原則 | 対応 |
|------|------|
| 統一・同一化 > 特殊独自 | メールと同様に「ゲスト画面に表示される」ことを入力欄付近で明示する。 |
| 具体的 > 一般 | 「入力して保存するとゲスト画面に表示されます。」と具体的に記載する。 |
| シンプル構造 > 複雑構造 | 1 文の但し書きを電話番号入力の直下に追加するのみとする。 |

---

## 3. 修正案

### 3.1 対象ファイル

- `frontend/src/views/admin/FacilitySettings.vue`

### 3.2 変更内容

**箇所**: 基本情報セクション内、電話番号の `Input` の直後（住所の `Input` の直前）。

**追加する要素**:

- 電話番号入力欄の直下に、1 文の但し書きを表示する。
- 文言: **「この電話番号は入力して保存するとゲスト画面に表示されます。」**
- スタイル: メールの但し書きと同系統の補足テキストとする。  
  - メールは「注意」のため `text-amber-600 dark:text-amber-400` を使用。  
  - 電話は「案内」のため、既存の補足文（例: ページ説明）に合わせて  
    `text-xs text-gray-600 dark:text-gray-400` とする。

**挿入イメージ**（該当部分のみ）:

```vue
<Input
  v-model="formData.phone"
  type="tel"
  label="電話番号"
  placeholder="例: 090-1234-5678"
  :maxlength="50"
  :error="errors.phone"
/>
<p class="text-xs text-gray-600 dark:text-gray-400">
  この電話番号は入力して保存するとゲスト画面に表示されます。
</p>
<Input
  v-model="formData.address"
  ...
/>
```

### 3.3 補足

- 電話番号は「表示する/しない」のスイッチはなく、**入力されていればゲスト画面に表示**される仕様のため、常時表示の但し書きでよい。
- マニュアル（Manual.vue）の「6.2 基本情報設定」で電話番号の説明に、同様の 1 文（「入力するとゲスト画面に表示されます」など）を追記してもよい（任意）。
