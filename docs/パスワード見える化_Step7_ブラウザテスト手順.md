## 3. 確認項目（各画面で実施）

### 必須

1. **目アイコン表示**: パスワード入力欄の**右側**に目アイコン（トグルボタン）が表示されていること。
2. **クリックで表示**: トグルをクリックすると、入力内容が**表示**（マスク解除）されること。
3. **クリックで非表示**: 再度クリックすると、入力内容が**マスク**（•••）に戻ること。

### 任意（余裕があれば）

4. **他入力欄**: メール・施設名など、トグルを付けていない Input は従来どおりであること。
5. **エラー時**: パスワード欄でエラー時、右側は**トグルボタンのみ**で、エラーメッセージは**下**に表示されること（ログインで誤パスワード等で確認可）。
