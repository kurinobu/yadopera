---

## 2. ブラウザテスト手順（FAQモーダル）

1. **管理者でログイン**し、**FAQ管理**画面を開く。
2. **「+ 新規FAQ追加」**をクリックし、FAQ追加モーダルを開く。
3. **何も入力しない状態**で、モーダル外（背景の暗い部分）をクリック  
   → **モーダルが閉じること**（従来どおり）。
4. 再度「+ 新規FAQ追加」を開き、**質問文または回答文に1文字以上入力**する。
5. モーダル外をクリック  
   → **モーダルは閉じず**、**「入力内容を残すには『作成』を、破棄するには『キャンセル』をクリックしてください。」**の警告バナーが表示されること。
6. **ESCキー**を押す  
   → 同様に**閉じず**、警告バナーが表示されたままであること。
7. ヘッダーの**×ボタン**をクリック  
   → 同様に**閉じず**、警告バナーが表示されたままであること。
8. **「キャンセル」**をクリック  
   → モーダルが閉じること。
9. 再度「+ 新規FAQ追加」で入力し、**「作成」**で保存  
   → モーダルが閉じ、一覧に反映されること。
10. **編集モード**で既存FAQを開き、内容を変更してから外側クリック  
    → 閉じず、**「更新」**を促す警告が出ること。
